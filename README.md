# PrivacyCam - SNS画像加工アプリ

AIを活用して、SNSにアップロードする画像のプライバシー保護加工を行うWebアプリです。
バージョン2.0では、より柔軟な編集が可能な「インタラクティブエディタ」機能を搭載しました。

## 新機能: インタラクティブ編集 & UX向上
- **自動検出**: 画像を読み込むと自動で顔を検出し、デフォルトでモザイクをかけます。
- **モザイク強度調整**: スライダーでモザイクの粗さを自由に調整できます。
- **ポジティブスタンプ & ランダム**: 👹などの怖いスタンプを排除し、可愛い動物やキラキラ系を追加。ランダムモードではクリックごとに違うスタンプが出ます。
- **手動追加エリア**: 画面をドラッグすることで、検出されなかった場所や隠したい背景などに手動で加工エリアを追加できます。
- **並び替え & 連番保存**: サイドバーの画像をドラッグして順番を入れ替えられます。保存時は上から順に連番でダウンロードできます。
- **ファイル名指定**: ダウンロード時のファイル名を自由に指定できます。

## 使い方

### 1. 起動
ターミナルで以下のコマンドを実行し、ローカルサーバーを立ち上げてください。
（ブラウザのセキュリティ制限により、直接開くと動作しない場合があります）

```bash
python3 -m http.server 8000
```
その後、ブラウザで `http://localhost:8000` にアクセスします。

### 2. 操作方法
1. **画像の追加**: 画像ファイルを画面にドラッグ＆ドロップするか、「+ 追加」ボタンから選択します。
2. **ツールの選択**: 画面上部のツールバーで「モザイク(🌫️)」または「絵文字(☺️)」を選択します。
3. **加工の編集**:
   - **検出された顔**: 四角枠をクリックすると、選択中のツール効果（モザイク/絵文字）が適用されます。もう一度クリックすると変更/解除できます。
   - **手動エリア**: 画像上の好きな場所をドラッグすると、その範囲に加工を追加できます。
4. **保存**: 「保存」ボタンを押すと、現在の加工状態で画像をダウンロードできます。
