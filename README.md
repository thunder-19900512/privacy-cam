# PrivacyCam - SNS画像加工アプリ

AIを活用して、SNSにアップロードする画像のプライバシー保護加工を**ブラウザ上ですべて完結**させるWebアプリです。
画像データは外部サーバーに送信されないため、安心して家族や友人の写真を加工できます。

## 主要機能

### 1. AIによる自動顔検出
- 画像を読み込むと自動で顔を検出し、デフォルトで加工を適用します。
- **HEIC形式対応**: iPhoneで撮影されたHEIC/HEIF形式の画像もそのまま読み込み可能です。

### 2. インタラクティブ編集
- **モザイク & 絵文字**: 🌫️モザイク（強度調整可能）と☺️絵文字（カスタム入力・ランダム切り替え可能）を自由に使い分けられます。
- **手動選択エリア**: 検出されなかった箇所や背景などは、ドラッグで自由に加工範囲を追加・削除できます。
- **ドラッグ&ドロップ操作**: 加工後の顔をクリックするだけで、効果の切り替えや解除が可能です。

### 3. 高度な一括処理
- **並び替え**: サイドバーで画像をドラッグして順番を入れ替えられます。
- **連番ダウンロード**: 保存時のファイル名を指定でき、一括保存時は自動で連番が付与されます。
- **サイズ制限機能**: 指定したファイル容量（MB）以下になるよう、自動でリサイズして保存できます。

### 4. 信頼のプライバシー
- **100% クライアントサイド処理**: すべての画像処理はブラウザ内で行われ、サーバーには一切データが残りません。
- **利用規約・プライバシーポリシー**: アプリ下部のリンクからいつでも詳細を確認できます。

## 使い方

### 1. 起動
ブラウザのセキュリティ制限（CORSなど）により、直接HTMLファイルを開くと動作しない場合があります。
以下のコマンドなどでローカルサーバーを立ち上げることをお勧めします。

```bash
# Pythonを使用する場合
python3 -m http.server 8000
```
その後、ブラウザで `http://localhost:8000` にアクセスします。

### 2. 操作
1. **画像を追加**: 画像をドロップするか「画像を選択」ボタンから追加します。
2. **加工を選択**: 上部のツールバーで効果を選び、画像上の枠をクリックまたはドラッグして適用します。
3. **保存**: 「保存」または「すべて保存」で、加工された画像をダウンロードします。

## 技術スタック
- **UI/UX**: HTML5 / CSS3 (Glassmorphism & Retro Theme Support)
- **Engine**: JavaScript (Vanilla) + HTML5 Canvas
- **AI**: [face-api.js](https://github.com/vladmandic/face-api) (SSD Mobilenet v1)
- **Utility**: [heic2any](https://github.com/alexcorvi/heic2any) (HEIC to JPEG conversion)
